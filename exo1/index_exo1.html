LE BINAIRE ET L'HEXADECIMAL

Système binaire

Le système binaire est le système de numération utilisant la base 2. On nomme couramment bit les chiffres de la numération binaire positionnelle.Un bit peut prendre deux valeurs, notées par convention 0 et 1.Le système binaire est utile pour représenter le fonctionnement de l'électronique numérique utilisée dans les ordinateurs.Il est donc utilisé par les langages de programmation de bas niveau.

Code binaire

11111001000

Explication: système de numération utilisant deux valeurs, le 0 et le 1. On commence par la droite, chaque chiffre ajoutera 1 à l'exposant de 2.

Code hexadécimal

7C8

Explication: on sépare par bloc de 4 le binaire (on ajoute des 0 si nécessaire au premier bloc en partant de la gauche). On obtient donc :

0111 1100 1000

On fait la somme des 1 par rapport à 2 exposant n. On aura donc 0111 = 0 + 4 + 2 + 1 = 7, 1100 = 8 + 4 + 0 + 0 = 12.

Jusqu'à 9 l'hexadécimal utilise les nombres décimaux de base.

Au dessus on part sur l'alphabet en commençant par A.

7 = 7 et 12 = C
